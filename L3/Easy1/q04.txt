numbers.delete_at(1) deletes element 1 of numbers (e.g., the element with a
value of 2 in this case). The return value of the expression is 2 in this case,
and the end value of numbers is [1, 3, 4, 5]

numbers.delete(1) deletes all elements of numbers that have a value of 1
(e.g., just the element with index 0 in this case). The return value of the
expression is 1, and the end value of numbers i [2, 3, 4, 5]
